# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 提供在此代码库中工作的指导。

> **重要提示**: 每次执行 Python 代码前，必须先激活虚拟环境 `conda activate deep-research`

## 项目目的

学习 Claude Code 底层原理及最佳实践的研究项目。

## 开发环境

**后端 (Python)**
- 虚拟环境: `conda activate deep-research`
- 代码规范: 遵循 PEP 8 标准

**前端**
- 语言: TypeScript

## 开发命令

```bash
# 激活 Python 虚拟环境
conda activate deep-research

# 安装依赖（当存在 requirements.txt 时）
pip install -r requirements.txt
```
